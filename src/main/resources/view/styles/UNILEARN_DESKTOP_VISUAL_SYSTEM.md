# UniLearn Desktop Visual System

## Identity Principles
- Keep UniLearn naming, educational language, and role vocabulary.
- Use desktop-first workspace structure: left rail navigation, contextual top header, center module canvas.
- Prioritize long-session readability with soft contrast and generous spacing.
- Preserve brand identity without mirroring web layout one-to-one.

## Layout Pattern
- One AppShell root in a single stage.
- Left rail: global module navigation and logout.
- Top header: context title, subtitle, current user badge.
- Center: dynamic module content (Login, Home, Users, Profile, Security).

## Color Direction
- Primary rail: deep academic blue (#0F2F4D -> #123A5D).
- Action blue: #1D6FA5 / hover #2482BF.
- Surface: white cards over light neutral workspace.
- Semantic danger: #8F2D2D.

## Typography & Rhythm
- Typeface: Segoe UI / Noto Sans fallback stack.
- Headings: strong, compact hierarchy.
- Card spacing: 16px internal, 12-16px external rhythm.
- Rounded corners for interactive elements and cards to reduce density fatigue.

## Motion & Interaction
- Hover color lift on interactive controls.
- Minimal transitions to keep desktop workflows fast.
- Use modal windows only for short focused forms when necessary.
